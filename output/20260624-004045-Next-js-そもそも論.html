1

そもそもNext.jsって何者？

Q
Reactとどう違うの？
A
Reactは「UIを作るためのライブラリ」。画面のパーツを作ることに特化しています。
一方Next.jsは、Reactを土台に「Webサービス全体を作るフレームワーク」です。

料理に例えると：
React ＝ 包丁（部品を切り出す道具）
Next.js ＝ キッチン全体（調理台・冷蔵庫・レンジまで揃っている）
React だけ

⚛️ Reactのみ

・UIの描画だけ
・ルーティング（画面遷移）は自分で設定
・SEOが弱い（最初にHTMLが空）
・パフォーマンス最適化も自前

Next.js

▲ Next.js（React入り）

・ファイルを置くだけでルーティング完成
・SEOに強いHTML生成が選べる
・画像最適化・キャッシュ自動対応
・バックエンドAPIも同じプロジェクト内に
