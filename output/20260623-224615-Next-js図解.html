🔍 Next.js とは

React + サーバー機能

Vercel製のReactフレームワーク。
SSR・SSG・APIルートを標準搭載し、
フロントからバックまで一元管理できる。

🖥 レンダリング方式
SSG

静的生成

ビルド時にHTMLを生成。高速配信に最適。

⏱ ビルド時
SSR

サーバー描画

リクエストごとにサーバーでHTML生成。

⏱ リクエスト時
ISR

増分再生成

一定間隔でページを再生成。SSGとSSRの中間。

⏱ 定期更新
CSR

クライアント描画

ブラウザ側でReactが動作する従来方式。
