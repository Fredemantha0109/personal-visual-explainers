🔗 API基礎ガイド

APIとは
Application Programming Interface
アプリケーション同士がデータを やり取りするための「約束された通信手段」
基本の流れ
📱 クライアント
リクエスト送信
↓
🖥️ サーバー
データ処理
↓
📨 レスポンス
データ返却
よくある使用例
✅ 天気アプリ → 天気データAPI
✅ 地図アプリ → 位置情報API
✅ SNS → 投稿データAPI
主なHTTPメソッド
GET → データ取得
POST → データ送信
PUT → データ更新
DELETE → データ削除

🎯 リクエスト→処理→レスポンスのサイクル
🎯 統一フォーマットで確実なやり取り
