⚛️ React基礎図解

1. JSX（JavaScriptとHTMLの融合）

JavaScriptの中にHTML構文を書ける特別な記法。ブラウザが理解できるJavaScriptに変換される。

const Button = () => {
return <button>クリック</button>;
};

↓ コンパイルされると React.createElement() に変換される

2. コンポーネント（再利用可能な部品）

UIを部品単位で分割・管理する。関数またはクラスで定義。

const Card = (props) => {
return <div>{props.title}</div>;
};
  • 関数型：シンプルで主流（hooks対応）
  • クラス型：古い書き方（ライフサイクルメソッド使用）

3. State（変動するデータ）

コンポーネントが保持するデータ。Stateが変わると画面が自動再レンダリングされる。

const [count, setCount] = useState(0);
<button onClick={() => setCount(count + 1)}>
{count}
</button>

useState は hooks の代表例。関数型コンポーネントでstate管理を実現。

4. Reactのデータフロー（単方向）

親コンポーネント（State保持）
↓ Props で下へ渡す
子コンポーネント（Props受け取り）
↑ コールバック関数で上へ通知
親が State を更新 → 再レンダリング

5. Reactが高速な理由

  • Virtual DOM：Reactが仮想的なDOMを保持。変更部分だけを実DOM に反映
  • 差分検出（Diffing）：前後の状態を比較。必要な部分だけ更新
  • バッチ処理：複数の更新を効率的にまとめて反映
