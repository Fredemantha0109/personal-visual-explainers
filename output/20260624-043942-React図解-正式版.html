⚛ React 図解

コンポーネント・状態・データフロー・Virtual DOM をひと目で理解する

① コンポーネントツリー
② 主要概念
🧩
コンポーネント
UIの部品単位。関数として定義し、JSXを返す。再利用・組み合わせが可能。
function Hello({ name }) { return <h1>{name}</h1>; }
🔄
State（状態）
コンポーネント内部のデータ。変化するとUIが自動で再レンダリングされる。
const [count, setCount] = useState(0); setCount(count + 1);
📦
Props（プロパティ）
親から子へ渡すデータ。読み取り専用で、子は直接変更できない。
<Button label="送信" onClick={fn} />
⚡
useEffect
API取得・タイマー等の副作用処理。依存配列で実行タイミングを制御する。
useEffect(() => { fetchData(); }, [id]);
🌐
Context
Propsを経由せず、離れたコンポーネントへデータをグローバルに共有する。
const ctx = useContext(ThemeCtx); // 深い子でも取得可
🗂
JSX
JavaScriptの中にHTML風の構文を書ける記法。ビルド時にJS関数へ変換される。
// JSX → JS React.createElement( 'div', null, 'Hello' )
